# B2B 外贸主动开发系统（基于 LLM + Multi-Agent System）

本项目旨在构建一套基于 **大语言模型（LLM）** 和 **多代理协作（Multi-Agent System）** 的 B2B 外贸主动开发系统。系统将彻底替代传统人工“搜客户-导表格-群发邮件”的低效模式，实现从泛搜到精准触达的闭环自动化。

---

## 核心架构设计

系统分为五个战略阶段，通过多个专项 Agent 协同工作：

---

### 第一阶段：由面到点精准搜索

- **输入端**：核心产品关键词或参考竞对域名。  
- **Agent 1 (Searcher)**：调用 Google Search API 进行全网批量泛搜。  
- **智能清洗**：自动排除 B2B 平台、广告、黄页等噪音，输出高质量目标企业官网列表。  

---

### 第二阶段：深度解析与穿透

- **Agent 2 (Parser)**：利用 Firecrawl 解析官网内容，评估业务匹配度。  
- **决策者定位**：调用 Apollo / Proxycurl API 精准锁定领英上的 Key Person。  
- **有效性验证**：通过 NeverBounce 实时验证商业邮箱与手机号，未锁定的数据转入人工池辅助处理。  

---

### 第三阶段：情报调研与 1:1 定制内容

- **Agent 3 (Researcher)**：检索企业近期新闻、动态及财报。  
- **内容生成**：生成极具个性化的开发信与 WhatsApp 文案，杜绝模版化。  
- **数据持久化**：所有情报与文案写入 Supabase 状态机数据库。  

---

### 第四阶段：多渠道自动化触达

- **Day 1**：Agent 4 通过 Instantly API 发送首封定制化邮件。  
- **动态跟进**：若 3 天内无回复，系统自动触发多渠道策略。  
- **Day 4**：通过 Wati API 发送 WhatsApp 友好跟进信息。  

---

### 第五阶段：意向分拣与客服收网

- **Agent 5 (Closer)**：实时监听回复，进行上下文意向分类：  
  - **A 类（积极）**：自动发送 Calendly 预约链接，并即时通过飞书 / 钉钉通知销售。  
  - **B 类（培育）**：标记标签，自动转入 CRM 长线培育序列。  
  - **C 类（拒绝）**：加入全系统黑名单，停止一切触达。
